## React Native

- No html tags
    - div -> view, scrollview
    - p, span -> text
    - display flex -> default = column
    - button -> button, touchableOpacity
    - img -> image
        - **Note** images in react native default to 0 width and height
    - input -> textInput
    - onClick -> onPress
    - onChange -> onChangeText
- No CSS
    - Must use js styles
